# Estudos-Java

Repositório utilizado para centralizar minhas anotações, exemplos de código, exercícios e testes relacionados ao ecossistema Java.

O objetivo deste repositório é manter uma base organizada de estudos para consulta, revisão e prática contínua durante minha evolução como desenvolvedor Java.

---

# 📂 Organização

Os conteúdos estão organizados por módulos e tópicos para facilitar a navegação e futuras revisões.

Exemplo:

```text
modulo-01-fundamentos
modulo-02-poo
modulo-03-collections
modulo-04-jdbc
```
